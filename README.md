PunchCard time keeper:
This program can allow employees to log in with their employee number, view their time sheet, and punch in or out through the interface.

Getting Started:
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

Prerequisites:
Web browser at minimum

Installing:
None required.

Deployment:
Drop folder into hosted environment, access through index.html by browser, or paste into codepen.
